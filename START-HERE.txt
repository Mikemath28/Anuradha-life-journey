ANURADHA — A JOURNEY THROUGH LIFE
=================================

This folder is ready to upload as a GitHub project.

FASTEST START
-------------
1. Open a terminal in this folder.
2. Run: npm install
3. Run: npm run dev

TO PUT ON GITHUB
----------------
1. Create an empty GitHub repository.
2. Upload the contents of this folder, or use the git commands in README.md.
3. For GitHub Pages, go to Settings > Pages and choose GitHub Actions.
